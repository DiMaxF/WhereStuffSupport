About the App

WhereStuff is an iOS application designed to help you track and organize items stored in different locations. Never lose track of where you've stored your belongings again.

Main Features

  • Add items with photos - Capture or select images for your stored items
  • Organize by storage places - Create and manage multiple storage locations
  • Tagging system - Use tags to categorize and quickly find items
  • Search and filtering - Quickly locate items using powerful search and filter options
  • Data export - Export your data in PDF or CSV format for backup or sharing

Version: 1.0

Platform: iOS 17.0+

Privacy Policy

Last updated: January 2025

Data Collection

The app does not collect, transmit, or store personal data outside your device. All data (items, photos, places, tags) is stored locally using SwiftData. The app does not use analytics or tracking services.

Data Storage

All data is stored locally on your device. Data is not transmitted to external servers. Data is encrypted by iOS using standard device encryption.

Data Usage

Data is used only for app functionality. No data is shared with third parties.

User Rights

  • Export data at any time (Settings → Export Data)
  • Delete all data (Settings → Clear All Data)
  • Uninstalling the app deletes all data

Permissions

  • Camera: for taking photos of items
  • Photo Library: for selecting images of items

Permissions are requested only when needed for app functionality.

Policy Changes

Any changes to this privacy policy will be communicated in the app and updated on this page. The "Last updated" date will reflect the most recent changes.
